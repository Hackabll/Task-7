# Task-7
Open Google Chrome’s Extension Manager

1. Open Google Chrome.


2. Click the three dots (⋮) in the top-right corner.


3. Hover over Extensions.


4. Click Manage Extensions.

Direct link: chrome://extensions/





---

2. Review All Installed Extensions

Inside the Extensions page:

You will see tiles for every extension installed.

Click Details on each extension to view:

What it does

Permissions

Version

Site access




---

3. Check Permissions and Reviews

For each extension, check:

✔ Permissions (very important)

Dangerous permissions include:

“Read and change all your data on all websites”

Access to clipboard

Access to browsing history


If an extension doesn’t need such permissions for its purpose → suspicious.

✔ Reviews

1. Click View in Chrome Web Store (inside the extension’s Details page).


2. Check:

Ratings

Number of users

Recent reviews mentioning malware, ads, pop-ups

Trusted developer? (Google, Microsoft, Grammarly, etc.)





---

4. Identify Unused or Suspicious Extensions

Flag extensions if:

You don’t remember installing them

No verified developer

Excessive permissions

Low rating

Reports of malware

Free utilities with generic names

You never use them



---

5. Remove Suspicious or Unneeded Extensions

1. Go to chrome://extensions/


2. Click Remove on the extension


3. Confirm removal




---

6. Restart Chrome

1. Close all Chrome windows


2. Reopen Chrome


3. Check:

Is it faster?

Are pop-ups gone?

Are redirects gone?





---

7. Research How Malicious Extensions Harm Users

Common dangers:

Stealing passwords

Tracking browsing

Injecting ads

Redirecting websites

Collecting personal data

Screenshot capturing

Selling user activity to trackers


(Search terms: “Chrome malicious extension examples”, “browser extension attack”.)


---

8. Document Everything

Create a report including:

Steps taken

Extensions reviewed

Extensions removed

Observed results
